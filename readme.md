## Homework_11
### Дебаг, тестирование кода.
Пример задачи из данного блока: 

```
Найти ошибку(и).
Функция calc(a, b, operation). 
Описание входных параметров:
    1. Первое число;
    2. Второе число;
    3. Действие над ними:
        + Сложить
        - Вычесть
        * Умножить
        / Разделить
        В остальных случаях функция должна возвращать "Операция не поддерживается"

def calc(a, b, operation):
    result = 'Операция не поддерживается'

    if operation == '+':
        result = a + b
    elif operation == '-':
        result = a - b
    elif operation == '*':
        result = a * b
    elif operation == '/':
        if b is not 0:
            result = a // b
        else:
            result = 'Деление на 0!'

    return result
```
